# CliniNote

Monorepo for the CliniNote MVP.

## Workspaces

- `apps/web`: Angular standalone frontend with NgRx Signals Store and Atomic Design folders.
- `apps/api`: Node.js TypeScript API using hexagonal architecture.
- `mcp-servers/audio-transcription-server`: MCP server for audio transcription.
- `mcp-servers/clinical-note-server`: MCP server for clinical note generation prompts.

## Commands

```bash
npm install
npm run dev
npm run build
npm test
```
